## 왜 필요한가

마우스로 모든 걸 할 수 있지만, 반복 작업은 키보드가 압도적으로 빠릅니다. Figma, Linear, Notion 같은 도구들이 키보드 단축키로 차별화합니다.

## 구현

```typescript
useEffect(() => {
  function handleKeyDown(e: KeyboardEvent) {
    if (e.key === "k" && (e.metaKey || e.ctrlKey)) {
      e.preventDefault();
      openCommandPalette();
    }
  }

  document.addEventListener("keydown", handleKeyDown);
  return () => document.removeEventListener("keydown", handleKeyDown);
}, []);
```

## 고려사항

- **input/textarea 포커스 시 무시** — 타이핑 중 단축키가 발동하면 안 됨
- **운영체제 단축키와 충돌 방지** — `Cmd+W` 같은 건 쓰지 마세요
- **접근성** — `aria-keyshortcuts` 속성 활용
- **Discoverability** — `?`로 단축키 목록 표시

## 라이브러리

- `tinykeys` — 1KB, 간단한 API
- `hotkeys-js` — 더 많은 기능

> 단축키는 파워유저를 위한 선물입니다. 하지만 없이도 모든 기능이 동작해야 합니다.
