## 기본 헤더

```
Cache-Control: public, max-age=31536000, immutable
```

- `public` — CDN에서 캐시 가능
- `max-age` — 초 단위 TTL
- `immutable` — 재검증 요청도 보내지 않음

## 전략별 정리

### 정적 에셋 (JS, CSS, 이미지)

파일명에 해시가 포함되면 영구 캐시가 안전합니다.

```
Cache-Control: public, max-age=31536000, immutable
```

### HTML

항상 최신 버전을 제공해야 하므로 재검증합니다.

```
Cache-Control: no-cache
ETag: "abc123"
```

### API 응답

사용자별 데이터는 캐시하지 않습니다.

```
Cache-Control: private, no-store
```

> 캐시는 성능 최적화의 가장 큰 레버입니다. 하지만 잘못 쓰면 디버깅 지옥이 됩니다.
