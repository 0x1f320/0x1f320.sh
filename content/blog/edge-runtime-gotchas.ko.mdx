## Edge Runtime이란

V8 isolate 기반의 경량 런타임입니다. Vercel Edge Functions, Cloudflare Workers 등이 여기에 해당합니다. 콜드 스타트가 거의 없는 대신, Node.js API를 전부 쓸 수 없습니다.

## fs 모듈

```typescript
import fs from "node:fs"; // ❌ 동작하지 않음
```

파일 시스템이 없습니다. 빌드 타임에 번들에 포함시키거나 KV/R2 같은 스토리지를 써야 합니다.

## crypto

`node:crypto`는 안 되지만 Web Crypto API는 됩니다.

```typescript
// ❌
import { randomBytes } from "node:crypto";

// ✅
const array = new Uint8Array(16);
crypto.getRandomValues(array);
```

## 시간 제한

대부분의 Edge 런타임은 실행 시간이 제한됩니다. 긴 작업은 큐에 넣고 서버리스 함수에서 처리하세요.

> Edge는 빠르지만 만능이 아닙니다. 제약을 먼저 파악하세요.
