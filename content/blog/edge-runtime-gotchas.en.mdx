## What Is Edge Runtime

A lightweight runtime based on V8 isolates. Vercel Edge Functions and Cloudflare Workers fall into this category. Near-zero cold starts, but you can't use all Node.js APIs.

## fs Module

```typescript
import fs from "node:fs"; // ❌ Doesn't work
```

There's no filesystem. Bundle at build time or use KV/R2 storage.

## crypto

`node:crypto` doesn't work, but Web Crypto API does.

```typescript
// ❌
import { randomBytes } from "node:crypto";

// ✅
const array = new Uint8Array(16);
crypto.getRandomValues(array);
```

## Time Limits

Most Edge runtimes have execution time limits. Queue long tasks and handle them in serverless functions.

> Edge is fast, but not a silver bullet. Know the constraints first.
