## 문제

TypeScript의 타입은 컴파일 타임��만 존재합니다. 외부 API가 스키마를 바꾸면 `as` 캐스팅한 코드는 조용히 깨집니다.

```typescript
const data = await fetch("/api/users").then((r) => r.json()) as User[];
// API가 { users: User[] }로 바뀌면? 런타임 에러
```

## Zod 스키마

```typescript
import { z } from "zod";

const UserSchema = z.object({
  id: z.string(),
  name: z.string(),
  email: z.string().email(),
});

const UsersResponse = z.array(UserSchema);

type User = z.infer<typeof UserSchema>;
```

## 사용

```typescript
const raw = await fetch("/api/users").then((r) => r.json());
const users = UsersResponse.parse(raw);
// 스키마가 안 맞으면 여기서 에러 — 조용히 깨지지 않음
```
