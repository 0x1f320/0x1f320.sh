## 문제

모노레포에서 패키지마다 React 버전이 다릅니다.

```
packages/app: react@18.2.0
packages/ui:  react@18.3.1
packages/web: react@19.0.0
```

## pnpm catalogs

`pnpm-workspace.yaml`에 카탈로그를 정의합니다.

```yaml
catalog:
  react: ^19.0.0
  react-dom: ^19.0.0
  typescript: ^5.7.0
```

각 패키지의 `package.json`에서 참조합니다.

```json
{
  "dependencies": {
    "react": "catalog:",
    "react-dom": "catalog:"
  }
}
```

## 장점

- 버전 불일치 방지
- 업그레이드 시 한 곳만 수정
- `pnpm install` 시 카탈로그와 불일치하면 에러

> 버전 관리는 중앙에서, 사용은 분산으로.
