## Basic Headers

```
Cache-Control: public, max-age=31536000, immutable
```

- `public` — cacheable by CDN
- `max-age` — TTL in seconds
- `immutable` — skip revalidation requests entirely

## Strategies

### Static Assets (JS, CSS, Images)

If the filename contains a hash, permanent caching is safe.

```
Cache-Control: public, max-age=31536000, immutable
```

### HTML

Always serve the latest version — revalidate.

```
Cache-Control: no-cache
ETag: "abc123"
```

### API Responses

Don't cache user-specific data.

```
Cache-Control: private, no-store
```

> Caching is the biggest performance lever. But misconfigured caching is a debugging nightmare.
