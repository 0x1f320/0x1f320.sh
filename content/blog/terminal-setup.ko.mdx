## 셸

**zsh** + 플러그인 없음. Oh My Zsh는 느려서 뺐습니다. 필요한 건 직접 `.zshrc`에 넣습니다.

```bash
# 자주 쓰는 alias만
alias g="git"
alias gp="git push"
alias gl="git log --oneline -20"
alias dev="pnpm dev"
```

## 터미널 에뮬레이터

**Ghostty**. GPU 가속이라 빠르고, 설정이 단순합니다. 이전에는 iTerm2를 썼는데 돌아갈 이유가 없었습니다.

## 프롬프트

**Starship**. Rust로 만들어져서 빠르고, 설정 파일 하나로 끝납니다.

```toml
[character]
success_symbol = "[›](bold green)"
error_symbol = "[›](bold red)"

[git_branch]
format = "[$branch]($style) "

[nodejs]
disabled = true
```

## 기타 도구들

- **eza** — `ls` 대체. 아이콘, 색상, Git status 표시
- **zoxide** — `cd` 대체. 자주 가는 디렉토리를 기억함
- **fzf** — 퍼지 파인더. `Ctrl+R` 히스토리 검색에 필수
- **bat** — `cat` 대체. 신택스 하이라이팅
