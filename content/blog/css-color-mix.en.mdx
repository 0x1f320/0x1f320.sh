## The Problem

Every time you need a color variant in your design system, you add another token.

```css
--color-primary: #3b82f6;
--color-primary-light: #93bbfd;
--color-primary-dark: #1e40af;
--color-primary-10: rgba(59, 130, 246, 0.1);
```

## color-mix()

```css
.badge {
  background: color-mix(in srgb, var(--color-primary) 15%, transparent);
  color: var(--color-primary);
}

.button-hover {
  background: color-mix(in srgb, var(--color-primary), black 20%);
}
```

Derive variants from a single base color — fewer tokens to maintain.

## Browser Support

Supported in all modern browsers as of 2024. IE is dead, don't worry about it.
